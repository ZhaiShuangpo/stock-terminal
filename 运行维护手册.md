# 大A盯盘终端 - 运行维护手册

## 🚀 快速开始 (一键看盘)
在终端中执行以下命令：
```bash
cd /Users/zhai/Documents/Stock
sh start_all.sh
```
然后访问浏览器：[http://localhost:5173](http://localhost:5173)

---

## 🛑 彻底关闭 (下班收工)
执行以下命令杀死所有相关进程：
```bash
lsof -t -i:8000 | xargs kill -9
lsof -t -i:5173 | xargs kill -9
```

---

## 🛠️ 分步调试模式
如果想查看程序报错，可以手动启动：

### 1. 后端 (Data Engine)
```bash
cd backend
source venv/bin/activate
python main.py
```

### 2. 前端 (Web UI)
```bash
cd frontend
npm run dev
```

---

## 📁 目录结构说明
- `backend/`: Python FastAPI 服务，负责数据抓取和 Gemini AI 交互。
- `frontend/`: React TypeScript 项目，负责界面展示和拖拽交互。
- `优化后功能需求清单.md`: 项目的设计初衷与详细功能列表。

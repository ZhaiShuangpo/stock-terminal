# 大A Web端盯盘助手 - 功能需求与特性清单

## 🎯 核心定位
打造一个专为A股投资者设计的**现代化、高信息密度、跨平台**的 Web 盯盘终端。结合大模型（Gemini）能力，提供从“行情监控”到“异动归因与智能推演”的一站式投研辅助体验。

---

## 💡 用户视角下的痛点与 Web 端优化方案

### 1. 数据实时性与推送机制 (Web端核心架构)
* **痛点：** 传统网页频繁刷新导致体验卡顿，且客户端直接频繁请求容易被行情 API 封禁。
* **优化：** 
  * 采用 **WebSocket / SSE (Server-Sent Events)** 实现毫秒级数据推送，界面平滑无刷新更新。
  * 后端服务器统一获取多数据源（腾讯、新浪、东财），实现**自动轮询与容灾切换**，分发给 Web 客户端。
  * 状态栏实时显示“连接状态”与“延迟指示器”（如：`🟢 已连接 | 延迟: 15ms`）。

### 2. 交互体验与信息层级 (可视化升级)
* **痛点：** CLI 终端表达能力受限，无法直观展示复杂的 K 线和技术图表。
* **优化：** 
  * 引入专业级轻量图表库（如 TradingView Lightweight Charts），支持实时分时图、K线图秒级切换。
  * 采用**响应式网格布局 (Grid/Masonry)**，用户可自由拖拽、缩放、组合不同板块（如：自选股列表区、分时图区、资金流向面板、AI分析侧边栏）。

### 3. 多终端协同与隐私安全
* **痛点：** 换电脑后需要重新配置自选股和预警规则；云端同步可能泄露隐私。
* **优化：** 
  * 提供轻量级配置导出/导入，或基于 Web3 / 去中心化 / 纯本地 LocalStorage 存储机制。
  * AI 交互所需的数据（API Key）及用户的敏感持仓配置仅保存在本地浏览器缓存中，绝不上传至中央服务器。

## 📋 全新 Web 端功能需求详细清单 (V3.0)

### 模块一：全景视界（现代化监控看板）
1. **高密度数据表格：** 
   * 严格遵循A股习惯（涨红、跌绿、平盘灰白），支持护眼的**暗黑模式 (Dark Mode)** 与明亮模式一键切换。
   * 表头支持自定义拖拽排序、自由显示/隐藏（如：换手率、量比、市盈率、主力净流入、封单额）。
2. **多维图表联动与悬浮态：** 
   * 列表内支持 **Sparkline 迷你趋势图**。
   * 鼠标悬浮 (Hover) 股票名称即显示详细的今日分时走势缩略图。
   * 点击单只股票，侧边栏或底部抽屉无缝划出详细信息（包含五档/十档盘口、内外盘比例、大单分布）。
3. **全品种支持矩阵：** 
   * 无缝融合 A股正股、**可转债（特殊标记 T+0 及 20% 涨跌幅临停规则）**、ETF 基金及主力股指期货。
4. **办公防摸鱼 (Web 版 Boss Key)：** 
   * **伪装模式：** 提供一键皮肤切换功能，将界面伪装成类似 Excel 电子表格、VS Code 代码编辑器或纯黑白极简文本列表。
   * 快捷键（如 `Esc` 或双击空白处）极速隐藏红绿色彩和敏感列。

### 模块二：毫秒级策略预警系统
1. **可视化预警规则配置器：**
   * 提供直观的条件配置弹窗，支持逻辑组合：例如 `(价格 > X 且 涨幅 > Y%) 或 (5分钟涨跌幅绝对值 > 3%)`。
   * 支持特定品种告警（如：正股涨停且对应转债溢价率 < X% 时提示套利）。
2. **全方位触达提醒流：** 
   * **浏览器原生通知 (Web Notifications)** 及自定义提示音（不同告警级别匹配不同音效）。
   * 界面侧边栏设置 **“预警信息流 (Live Feed)”**，像微博时间线一样滚动展示触发的异动告警。
   * 系统后端支持扫码绑定微信、Telegram、钉钉机器人等第三方推送。

### 模块三：Gemini AI 智能投研辅助 (原生深度融合)
1. **智能异动悬浮窗 (AI 极速归因)：** 
   * 当个股触发异动预警时，后台静默抓取全网新闻/资金异动，并在列表中该股旁亮起“💡灵感图标”。
   * 用户鼠标悬浮即可查看 AI 提炼的一句话归因（如：`[AI] 受昨夜英伟达大涨及产业链订单追加传闻刺激，板块资金大幅流入。`）
2. **长文公告/研报 “太长不看 (TL;DR)”：** 
   * 个股详情页的“资讯/公告”面板集成“AI 极速总结”按钮。一键将百页长文财报提炼为 3 条核心要点（利好/利空/关键财务变动）。
3. **沉浸式 AI 盘后复盘实验室：** 
   * 提供独立的“盘后复盘”视图。
   * 一键生成图文并茂的复盘简报，总结今日账户/自选股表现、主力资金攻击方向，并基于技术指标（MACD/KDJ/均线）推演次日剧本。

### 模块四：极客级交互与高阶扩展
1. **全键盘操控 (Shortcut 优先理念)：** 
   * `Ctrl/Cmd + K`: 呼出全局搜索/命令面板（Omnibar），快速添加自选股、搜索资讯或直达某项功能。
   * `1-9`: 快速在不同的自选股分组（主线、高息、防守、转债）间切换标签页。
   * `Space / Enter`: 展开/收起选中股票的详情侧边栏。
2. **自定义工作流布局 (Workspace)：** 
   * 允许用户保存并快速切换多套 UI 布局方案（例如：“早盘激进盯盘布局”侧重分时和短线异动，“盘中静默监控布局”侧重视角隐蔽）。
3. **开放生态与自动化扩展准备：** 
   * 预留标准化的前端 Hook 和后端 API，便于未来对接自动化交易网关（如 QMT / Ptrade本地端），为实现“预警即交易”的条件单打下架构基础。
